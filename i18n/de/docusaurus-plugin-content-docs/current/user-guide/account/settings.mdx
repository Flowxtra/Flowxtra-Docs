# Kontoeinstellungen

Passen Sie Ihre Flowxtra-Erfahrung an, indem Sie Ihre Kontoeinstellungen konfigurieren. Dieser Leitfaden behandelt alle verfügbaren Einstellungen und wie Sie sie für Ihren Arbeitsablauf optimieren.

## Zugriff auf Einstellungen

Navigieren Sie im Hauptmenü zu **Einstellungen** oder klicken Sie auf Ihr Profilbild und wählen Sie **Kontoeinstellungen**.

## Profileinstellungen

### Persönliche Informationen

Aktualisieren Sie Ihre persönlichen Daten:

- **Vollständiger Name**: Ihr Anzeigename
- **E-Mail-Adresse**: Primäre Kontakt-E-Mail
- **Telefonnummer**: Kontakttelefon (optional)
- **Jobtitel**: Ihre Rolle im Unternehmen
- **Profilbild**: Avatar hochladen oder ändern

<Callout type="info">
Ihre E-Mail-Adresse wird für Anmeldung und Benachrichtigungen verwendet. Eine Änderung erfordert eine Verifizierung.
</Callout>

### Zeitzone

Stellen Sie Ihre Zeitzone für korrekte Zeitstempel ein:

- Automatisch vom Browser erkennen
- Manuell aus Liste wählen
- Beeinflusst Zeitpläne für Stellenanzeigen
- Steuert Benachrichtigungszeiten

### Spracheinstellungen

Wählen Sie Ihre Oberflächensprache:

- Englisch
- Deutsch
- Arabisch (العربية)

## Benachrichtigungseinstellungen

### E-Mail-Benachrichtigungen

Steuern Sie, wann Sie E-Mails erhalten:

**Bewerbungsbenachrichtigungen**:
- Neue Bewerbung eingegangen
- Bewerbungsstatus geändert
- Nachricht von Kandidat erhalten

**Job-Benachrichtigungen**:
- Job erfolgreich veröffentlicht
- Job läuft bald ab
- Job-Leistungsberichte

**Team-Benachrichtigungen**:
- Neues Teammitglied hinzugefügt
- Kommentar von Teammitglied
- Zur Überprüfung eines Kandidaten zugewiesen

**System-Benachrichtigungen**:
- Kontosicherheitswarnungen
- Abrechnungs- und Abonnement-Updates
- Produkt-Updates und Funktionen

### In-App-Benachrichtigungen

Konfigurieren Sie Dashboard-Benachrichtigungen:

- Echtzeit-Benachrichtigungen
- Benachrichtigungston
- Desktop-Benachrichtigungen (erfordert Browser-Berechtigung)
- Benachrichtigungs-Badge am Symbol

### Benachrichtigungshäufigkeit

Wählen Sie, wie oft Sie Zusammenfassungs-E-Mails erhalten:

- **Echtzeit**: Sofortige Benachrichtigungen
- **Stündlich**: Gebündelt jede Stunde
- **Täglich**: Einmal täglich Zusammenfassung
- **Wöchentlich**: Wöchentliche Zusammenfassung
- **Nie**: E-Mail-Benachrichtigungen deaktivieren

## Sicherheitseinstellungen

### Passwortverwaltung

**Passwort ändern**:

1. Aktuelles Passwort eingeben
2. Neues Passwort eingeben
3. Neues Passwort bestätigen
4. Klicken Sie auf **Passwort aktualisieren**

Passwortanforderungen:
- Mindestens 8 Zeichen
- Ein Großbuchstabe
- Ein Kleinbuchstabe
- Eine Zahl
- Ein Sonderzeichen (empfohlen)

<Callout type="warning">
Ändern Sie Ihr Passwort sofort, wenn Sie unbefugten Zugriff vermuten.
</Callout>

### Zwei-Faktor-Authentifizierung (2FA)

Fügen Sie eine zusätzliche Sicherheitsebene hinzu:

**2FA aktivieren**:

1. Gehen Sie zu **Sicherheit** > **Zwei-Faktor-Authentifizierung**
2. Klicken Sie auf **2FA aktivieren**
3. Scannen Sie den QR-Code mit einer Authenticator-App
4. Geben Sie den Bestätigungscode ein
5. Speichern Sie die Backup-Codes

**Unterstützte Authenticator-Apps**:
- Google Authenticator
- Microsoft Authenticator
- Authy
- 1Password

**Backup-Codes**:
- Generieren Sie 10 Einmal-Backup-Codes
- Sicher offline speichern
- Verwenden, wenn Sie Zugriff auf Authenticator verlieren

### Aktive Sitzungen

Aktive Anmeldesitzungen anzeigen und verwalten:

- Aktuelle Sitzung (dieses Gerät)
- Andere aktive Sitzungen
- Zeitstempel der letzten Aktivität
- Geräte- und Standortinformationen

**Sitzungen widerrufen**:
- Klicken Sie neben einer Sitzung auf **Widerrufen**
- Verwenden Sie **Alle widerrufen**, um sich überall abzumelden
- Erfordert erneute Anmeldung auf allen Geräten

### Anmeldeverlauf

Überprüfen Sie die letzte Anmeldeaktivität:

- Datum und Uhrzeit
- IP-Adresse
- Gerät und Browser
- Standort (ungefähr)
- Status (Erfolg oder Fehler)

<Callout type="warning">
Wenn Sie verdächtige Anmeldeaktivitäten sehen, ändern Sie sofort Ihr Passwort und aktivieren Sie 2FA.
</Callout>

## Datenschutzeinstellungen

### Datensichtbarkeit

Steuern Sie, wer Ihre Informationen sehen kann:

**Profilsichtbarkeit**:
- Öffentlich: Für alle Benutzer sichtbar
- Team: Nur für Teammitglieder sichtbar
- Privat: Nur für Sie sichtbar

**Aktivitätssichtbarkeit**:
- Online-Status anzeigen
- Letzte aktive Zeit anzeigen
- Aktivität im Feed anzeigen

### Datenexport

Exportieren Sie Ihre Daten:

1. Gehen Sie zu **Datenschutz** > **Datenexport**
2. Wählen Sie die zu exportierenden Daten
3. Wählen Sie das Format (JSON, CSV, PDF)
4. Klicken Sie auf **Export anfordern**
5. Download-Link per E-Mail erhalten

Exportierte Daten enthalten:
- Profilinformationen
- Stellenanzeigen
- Bewerbungen
- Teamaktivität
- Einstellungen und Präferenzen

### Datenaufbewahrung

Konfigurieren Sie, wie lange Daten aufbewahrt werden:

- Aktive Jobs: Unbegrenzt
- Geschlossene Jobs: 1 Jahr (Standard)
- Bewerbungen: 2 Jahre (Standard)
- Archivierte Daten: 5 Jahre (Standard)

## Integrationseinstellungen

### API-Zugriff

API-Schlüssel und Zugriff verwalten:

**API-Schlüssel erstellen**:

1. Gehen Sie zu **Integrationen** > **API-Schlüssel**
2. Klicken Sie auf **Neuen Schlüssel erstellen**
3. Benennen Sie Ihren Schlüssel
4. Wählen Sie Berechtigungen
5. Kopieren und speichern Sie den Schlüssel sicher

<Callout type="warning">
API-Schlüssel werden nur einmal angezeigt. Speichern Sie sie sicher und teilen Sie sie niemals öffentlich.
</Callout>

**API-Schlüssel-Berechtigungen**:
- Jobs lesen
- Jobs schreiben
- Bewerbungen lesen
- Bewerbungen schreiben
- Webhooks verwalten

### Webhooks

Webhook-Endpunkte konfigurieren:

1. Klicken Sie auf **Webhook hinzufügen**
2. Geben Sie die Endpunkt-URL ein
3. Wählen Sie Ereignisse zum Auslösen
4. Fügen Sie Authentifizierungs-Header hinzu (optional)
5. Testen Sie den Webhook
6. Speichern Sie die Konfiguration

**Verfügbare Ereignisse**:
- application.created (Bewerbung erstellt)
- application.updated (Bewerbung aktualisiert)
- job.published (Job veröffentlicht)
- job.closed (Job geschlossen)
- candidate.hired (Kandidat eingestellt)

### Drittanbieter-Integrationen

Verbinden Sie externe Dienste:

**Verfügbare Integrationen**:
- Google Kalender
- Slack
- Microsoft Teams
- Zapier
- LinkedIn

## Präferenzen

### Dashboard-Anpassung

Passen Sie Ihr Dashboard an:

- Standardansicht (Jobs, Bewerbungen, Analysen)
- Widget-Anordnung
- Schnellaktionen
- Favoritenfilter

### E-Mail-Vorlagen

Passen Sie automatisierte E-Mails an:

- Bewerbungsbestätigung
- Einladung zum Vorstellungsgespräch
- Absagebenachrichtigung
- Angebotsschreiben

**Vorlagenvariablen**:
- `{candidate_name}` (Kandidatenname)
- `{job_title}` (Jobtitel)
- `{company_name}` (Firmenname)
- `{interview_date}` (Interviewdatum)

### Anzeigepräferenzen

Passen Sie die Oberflächeneinstellungen an:

- Design: Hell, Dunkel oder Automatisch
- Dichte: Komfortabel, Kompakt
- Seitenleiste: Erweitert oder Eingeklappt
- Schriftgröße: Klein, Mittel, Groß

## Recruitment-Einstellungen

Konfigurieren Sie Ihren Einstellungsworkflow und Ihre Prozesse, um den Einstellungsanforderungen Ihres Unternehmens gerecht zu werden.

### Organisation & Team

Verwalten Sie die Organisationsstruktur Ihres Unternehmens:

**Unternehmensdetails**:
- Firmenname, Logo und Beschreibung bearbeiten
- Unternehmens-Website-URL aktualisieren
- Unternehmensstandorte verwalten

**Teammitglieder**:
- Neue Benutzer hinzufügen (Recruiter, Hiring Manager)
- Rollen und Berechtigungen zuweisen
- Teamzugriffsstufen verwalten

**Büros**:
- Zweigstellenadressen verwalten
- Mehrere Bürostandorte hinzufügen
- Standardstandort für Jobs festlegen

**Vertragsunternehmen**:
- Kundenunternehmen verwalten (für Personalvermittlungen)
- Tochtergesellschaftsprofile handhaben
- Unternehmensübergreifende Workflows konfigurieren

<Callout type="info">
Wenn Sie für mehrere Unternehmen rekrutieren oder Tochtergesellschaften haben, können Sie alle deren Profile in diesem Abschnitt verwalten.
</Callout>

### Einstellungsprozess

Passen Sie an, wie Kandidaten durch Ihre Einstellungspipeline laufen:

**Pipeline-Editor**:
- Benutzerdefinierte Pipeline-Stufen erstellen (z. B. Bewerbung, Interview, Test, Angebot)
- Stufenspezifische Aktionen konfigurieren
- Automatisierte Workflows einrichten
- Abteilungsspezifische Pipelines erstellen

<Callout type="tip">
Sie können verschiedene Pipelines für verschiedene Abteilungen oder Jobtypen erstellen, um Ihren individuellen Einstellungsprozessen gerecht zu werden.
</Callout>

**E-Mail-Vorlagen**:
- Wiederverwendbare E-Mail-Vorlagen erstellen
- Absagen, Einladungen zu Vorstellungsgesprächen und Angebotsschreiben anpassen
- Dynamische Variablen wie `{candidate_name}`, `{job_title}`, `{company_name}` verwenden
- Vorschau vor dem Senden

Verfügbare Vorlagentypen:
- Bewerbungsbestätigung
- Einladung zum Vorstellungsgespräch
- Absagebenachrichtigung
- Angebotsschreiben
- Follow-up-E-Mails

**Bewertungsbögen**:
- Bewertungskriterien für Kandidaten erstellen
- Interviewbewertungen standardisieren
- Faire und konsistente Bewertung sicherstellen
- Feedback von Interviewern verfolgen

### Employer Branding

Passen Sie das Erscheinungsbild Ihres Unternehmens an, um Top-Talente anzuziehen:

**Job Widget**:
- Das Joblisten-Widget für Ihre Unternehmenswebsite anpassen
- Layoutstil wählen (Liste, Raster oder Karten)
- Farben und Schriftarten konfigurieren
- Einbettungscode für Ihre Website

**Unternehmensseite**:
- Eine öffentlich zugängliche Unternehmensprofilseite gestalten
- Unternehmenskultur und Werte präsentieren
- Fotos und Videos hinzufügen
- Mitarbeiter-Testimonials anzeigen

**Marken-Kit (Brand Kit)**:
- Unternehmensmarkenfarben festlegen
- Typografie und Schriftarten definieren
- Konsistente visuelle Identität sicherstellen
- Branding auf alle Stellenanzeigen anwenden

<Callout type="tip">
Eine starke Arbeitgebermarke kann die Bewerbungsraten um bis zu 50 % erhöhen. Nehmen Sie sich Zeit, um Ihre Unternehmensseite und Ihr Branding anzupassen.
</Callout>

### System & Konto

**Abrechnung & Abonnements**:
- Siehe [Abrechnung & Abonnement](/user-guide/account/subscription-billing)

**Benachrichtigungen**:
- Einstellungsbezogene Benachrichtigungen steuern
- Warnungen für Teamzusammenarbeit einrichten
- Bewerbungsbenachrichtigungen konfigurieren

**Benutzerdefinierte Domain**:
- Eine benutzerdefinierte Domain für Ihre Karriereseite einrichten (z. B. jobs.ihrefirma.com)
- DNS-Einstellungen konfigurieren
- HTTPS/SSL aktivieren

**Integrationen & API**:
- Mit externen Plattformen verbinden (Slack, Zoom, Microsoft Teams)
- API-Zugriff für benutzerdefinierte Integrationen einrichten
- Webhooks für Echtzeit-Updates konfigurieren

<Callout type="tip">
**Empfohlenes Setup**: Konfigurieren Sie Ihre Pipeline und E-Mail-Vorlagen, bevor Sie Ihren ersten Job veröffentlichen, um später Zeit zu sparen.
</Callout>

## Social-Media-Einstellungen

Verwalten Sie alle Social-Media-Konten Ihres Unternehmens an einem Ort für einheitliches Posten und Planen.

### Verbundene Konten

Anzeigen und Verwalten aller derzeit mit dem System verbundenen Social-Media-Konten:

**Unterstützte Plattformen**:
- Facebook-Seiten
- Instagram Business
- LinkedIn-Unternehmensseiten
- Twitter/X
- TikTok Business

Für jedes verbundene Konto sehen Sie:
- **Kontoname & Avatar**: Überprüfen Sie, ob das richtige Konto verbunden ist
- **Plattform-Symbol**: Visueller Indikator für den Plattformtyp
- **Verbindungsstatus**: Aktiv oder erneute Verbindung erforderlich

### Neues Konto verbinden

Um ein neues Social-Media-Konto hinzuzufügen:

1. Klicken Sie auf die Schaltfläche **Konto verbinden**
2. Wählen Sie die Plattform aus, die Sie verbinden möchten (z. B. Facebook-Seite)
3. Sie werden zur Anmeldeseite dieser Plattform weitergeleitet
4. Autorisieren Sie die Berechtigungen
5. Nach der Genehmigung werden Sie zurück zu Flowxtra geleitet
6. Das Konto erscheint in Ihrer Liste der verbundenen Konten

<Callout type="warning">
**Wichtig**: Stellen Sie für Seiten (Facebook, LinkedIn) sicher, dass Ihr persönliches Konto über **Admin**- oder **Editor**-Berechtigungen auf der Unternehmensseite verfügt, die Sie verwalten möchten.
</Callout>

### Konto erneut verbinden

Manchmal laufen Zugriffstoken aus Sicherheitsgründen ab (z. B. nach Änderung Ihres Passworts auf der Plattform):

**Wann eine erneute Verbindung erforderlich ist**:
- Token-Ablauf (normalerweise nach 60-90 Tagen)
- Passwortänderung auf der Plattform
- Berechtigungen widerrufen oder geändert

**So verbinden Sie sich erneut**:
1. Suchen Sie nach der Warnung zur erneuten Verbindung neben dem Konto
2. Klicken Sie auf die Schaltfläche **Erneut verbinden**
3. Befolgen Sie dieselben Autorisierungsschritte
4. Ihre vorherigen Daten und Einstellungen bleiben erhalten

<Callout type="info">
Das erneute Verbinden eines Kontos wirkt sich nicht auf Ihre zuvor veröffentlichten Beiträge oder geplanten Inhalte aus.
</Callout>

### Konto trennen

Um ein Konto zu entfernen und das Posten darauf zu beenden:

1. Suchen Sie das Konto in der Liste
2. Klicken Sie auf das **Löschen**-Symbol (Papierkorb)
3. Bestätigen Sie die Aktion im Popup-Dialog

<Callout type="warning">
**Hinweis**: Das Trennen eines Kontos löscht keine bereits auf der Plattform veröffentlichten Beiträge, verhindert jedoch, dass das System neue Beiträge an dieses Konto sendet.
</Callout>

### Verwalten mehrerer Konten

**Best Practices**:
- Verbinden Sie alle relevanten Unternehmens-Social-Media-Konten
- Überprüfen Sie regelmäßig den Verbindungsstatus
- Erneuern Sie Verbindungen, bevor sie ablaufen
- Weisen Sie Teammitglieder zu, um bestimmte Konten zu verwalten

**Kontoorganisation**:
- Gruppieren Sie Konten nach Marke oder Standort
- Kennzeichnen Sie Konten bei Unternehmen mit mehreren Standorten eindeutig
- Legen Sie Standardkonten für automatisches Posten fest

## Erweiterte Einstellungen

### Konto löschen

Konto dauerhaft löschen:

1. Gehen Sie zu **Erweitert** > **Konto löschen**
2. Überprüfen Sie, was gelöscht wird
3. Geben Sie Ihr Passwort ein
4. Tippen Sie "DELETE" zur Bestätigung
5. Klicken Sie auf **Konto dauerhaft löschen**

<Callout type="warning">
**Warnung**: Das Löschen des Kontos ist dauerhaft und kann nicht rückgängig gemacht werden. Alle Daten werden dauerhaft gelöscht.
</Callout>

Vor dem Löschen:
- Exportieren Sie Ihre Daten
- Kündigen Sie aktive Abonnements
- Schließen Sie alle Stellenanzeigen
- Benachrichtigen Sie Teammitglieder

### Datenmigration

Daten auf ein anderes Konto übertragen:

1. Kontaktieren Sie support@flowxtra.com
2. Geben Sie Quell- und Zielkonten an
3. Geben Sie die zu migrierenden Daten an
4. Migrationsanfrage bestätigen
5. Migrierte Daten überprüfen

## Tastaturkürzel

Aktivieren Sie Tastaturkürzel für schnellere Navigation:

**Globale Kürzel**:
- `Strg/Cmd + K`: Schnellsuche
- `Strg/Cmd + N`: Neuer Job
- `Strg/Cmd + ,`: Einstellungen
- `Strg/Cmd + /`: Kürzel anzeigen

**Navigation**:
- `G dann D`: Zum Dashboard
- `G dann J`: Zu Jobs
- `G dann A`: Zu Bewerbungen
- `G dann S`: Zu Einstellungen

Aktivieren Sie dies in **Präferenzen** > **Tastaturkürzel**.

## Fehlerbehebung

### Einstellungen werden nicht gespeichert

Wenn Einstellungen nicht gespeichert werden:

1. Internetverbindung prüfen
2. Seite aktualisieren
3. Browser-Cache leeren
4. Anderen Browser versuchen
5. Support kontaktieren

### E-Mail-Benachrichtigungen nicht erhalten

Überprüfen Sie diese Einstellungen:

1. E-Mail-Adresse verifizieren
2. Spam-/Junk-Ordner prüfen
3. Sicherstellen, dass Benachrichtigungen aktiviert sind
4. Benachrichtigungshäufigkeit prüfen
5. noreply@flowxtra.com auf die Whitelist setzen

### 2FA-Probleme

Wenn Sie nicht auf 2FA zugreifen können:

1. Backup-Codes verwenden
2. Support mit Kontoverifizierung kontaktieren
3. Alternative Verifizierung bereitstellen
4. 2FA nach Identitätsbestätigung zurücksetzen
