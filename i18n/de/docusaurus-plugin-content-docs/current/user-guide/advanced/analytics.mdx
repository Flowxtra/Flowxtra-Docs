# Analysen & Berichte

Datengesteuertes Recruiting führt zu besseren Ergebnissen. Das Analyse-Dashboard bietet Einblicke in Ihre Recruiting-Performance.

## Übersichts-Dashboard

Zugriff über **Berichte** in der Seitenleiste.

### Wichtige Kennzahlen

- **Job-Ansichten**: Gesamter Traffic auf Ihren Anzeigen.
- **Bewerbungsrate**: Prozentsatz der Betrachter, die sich bewerben.
- **Time to Hire**: Durchschnittliche Tage von "Job veröffentlicht" bis "Eingestellt".
- **Pipeline-Gesundheit**: Anzahl der aktiven Kandidaten in jeder Stufe.

## Quellen-Berichte

Woher kommen Ihre besten Kandidaten?

- **Organisch**: Google, Bing
- **Direkt**: Ihre Website / Widget
- **Sozial**: LinkedIn, Twitter
- **Jobbörsen**: Empfehlungen von Aggregatoren
- **Empfehlungen**: Mitarbeiter-Empfehlungslinks

Nutzen Sie dies, um zu entscheiden, wo Sie Ihr Werbebudget einsetzen.

## Berichte exportieren

Müssen Sie Daten mit Stakeholdern teilen?

1. Klicken Sie oben rechts auf **Exportieren**
2. Wählen Sie den **Zeitraum** (Letzte 30 Tage, Lfd. Jahr, Benutzerdefiniert)
3. Wählen Sie das **Format**: CSV oder PDF
4. Herunterladen

:::info Datenschutz
Berichte enthalten aggregierte Daten. Um Rohdaten von Kandidaten zu exportieren, verwenden Sie die Option "Kandidaten exportieren" im Job-Dashboard.
:::
