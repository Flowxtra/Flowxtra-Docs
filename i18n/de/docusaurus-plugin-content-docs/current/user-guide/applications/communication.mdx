# Kommunikation

Halten Sie Kandidaten mit integrierten Nachrichtenfunktionen auf dem Laufenden.

## E-Mail-Integration

Flowxtra synchronisiert sich mit E-Mails, sodass Sie Nachrichten an Kandidaten senden können, ohne das Dashboard zu verlassen.

### Eine E-Mail senden

1. Öffnen Sie ein Kandidatenprofil
2. Klicken Sie auf den Reiter **E-Mail**
3. Tippen Sie Ihre Nachricht oder wählen Sie eine **Vorlage**
4. Klicken Sie auf **Senden**

Der Kandidat erhält die E-Mail in seinem normalen Posteingang. Wenn er antwortet, erscheint dies in Flowxtra UND Ihrem E-Mail-Posteingang (falls verbunden).

## E-Mail-Vorlagen

Sparen Sie Zeit mit standardisierten Antworten.

### Vorlagen erstellen

Gehen Sie zu **Einstellungen** > **E-Mail-Vorlagen**, um Folgendes zu erstellen:

- Einladungen zum Vorstellungsgespräch
- Absageschreiben
- Angebotsschreiben
- Anfragen für "Weitere Informationen"

### Variablen verwenden

Personalisieren Sie Vorlagen automatisch:

- `{first_name}` (Vorname)
- `{job_title}` (Stellentitel)
- `{company_name}` (Firmenname)

## Interviews planen

### Manuelle Planung

Senden Sie eine E-Mail mit Terminvorschlägen.

### Kalender-Integration (Demnächst)

Verbinden Sie Google Kalender oder Outlook, damit Kandidaten verfügbare Termine direkt buchen können.

## Massennachrichten

Müssen Sie mehrere Kandidaten aktualisieren?

1. Gehen Sie zur **Listenansicht**
2. Wählen Sie Kandidaten aus
3. Klicken Sie auf **Ausgewählte mailen**
4. Wählen Sie eine Vorlage (z. B. "Leider wurde die Stelle besetzt")
5. Senden

:::info Datenschutz
Massen-E-Mails werden als BCC oder individuelle Nachrichten gesendet. Kandidaten sehen niemals die E-Mail-Adressen der anderen.
:::
