# Statusverfolgung

Erfahren Sie, wie Sie Ihre Einstellungspipeline anpassen und verwalten.

## Pipeline-Stufen

Die Pipeline definiert die Schritte, die ein Kandidat durchläuft. Sie können diese Stufen an den tatsächlichen Einstellungsprozess Ihres Unternehmens anpassen.

### Stufen anpassen

1. Gehen Sie zu **Einstellungen** > **Pipeline**
2. **Stufe hinzufügen**: Erstellen Sie eine neue Spalte (z. B. "Tech-Assessment", "Kultur-Interview")
3. **Neu anordnen**: Ziehen Sie Stufen, um ihre Reihenfolge zu ändern
4. **Umbenennen**: Bearbeiten Sie bestehende Stufennamen
5. **Löschen**: Entfernen Sie ungenutzte Stufen (Systemstufen wie "Neu" oder "Eingestellt" können nicht gelöscht werden)

## Stufen-Automatisierungen

Sie können Aktionen auslösen, wenn ein Kandidat in eine Stufe eintritt:

- **E-Mail senden**: Senden Sie automatisch eine "Danke für Ihre Bewerbung"-E-Mail beim Wechsel zu "Neu".
- **Teammitglied zuweisen**: Weisen Sie den Head of Engineering zu, wenn der Wechsel zu "Tech-Interview" erfolgt.
- **Info anfordern**: Senden Sie einen Link zu einem Fragebogen beim Wechsel zu "Screening".

### Automatisierung einrichten

1. Klicken Sie auf das **Zahnrad-Symbol** in der Kopfzeile einer Pipeline-Spalte
2. Wählen Sie **Automatisierung hinzufügen**
3. Wählen Sie Auslöser und Aktion

:::tip Best Practice
Automatisieren Sie Absage-E-Mails nicht sofort, wenn Sie einen Kandidaten auf "Abgelehnt" setzen. Es ist besser, sie zu überprüfen und stapelweise oder mit Verzögerung zu versenden, um eine menschliche Note zu bewahren.
:::
