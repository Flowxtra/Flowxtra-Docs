# Available MCP Tools

The Flowxtra MCP server exposes the following tools to your AI agent:

## `list_jobs`
List active job postings.
- **Args**: `status` (optional), `limit` (optional)

## `get_job_details`
Get full details like description and requirements.
- **Args**: `job_id`

## `list_applications`
Get candidates for a job.
- **Args**: `job_id`

## `get_candidate_profile`
Read resume text and Q&A.
- **Args**: `application_id`

## `add_internal_note`
Let the AI write a summary or note on a candidate.
- **Args**: `application_id`, `note_content`

## `send_email_to_candidate`
Send email to an applicant.
- **Args**: `application_id`, `subject`, `message`

## `create_job_post`
Create a new job posting quickly.
- **Args**: `title`, `description`, `location`, `employment_type`, etc.

## Example Prompts
> "List my 'Marketing Manager' job applications and summarize the key experience from their resumes."

> "Send an interview invitation email to John Smith for the Marketing Manager position."

> "Create a new job post for a Senior Software Engineer position in Berlin."

<Callout type="info">
**Note**: The AI does **not** make matching or ranking decisions. All candidate evaluation and selection decisions are made by the hiring manager. The AI only helps with listing, summarizing information, sending emails, and creating job posts faster.
</Callout>
