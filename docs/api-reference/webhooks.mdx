# Webhooks

:::info Coming Soon
Webhooks are planned for a future release. The data model is ready but the API endpoints are not yet available.
:::

Webhooks will allow you to receive real-time notifications when events happen in your Flowxtra account.

## Planned Events

| Event | Description |
|---|---|
| `application.created` | New candidate application received |
| `application.stage_changed` | Candidate moved to a different pipeline stage |
| `application.rejected` | Candidate rejected |
| `job.published` | Job posted/published |
| `job.closed` | Job closed or archived |
| `candidate.hired` | Candidate marked as hired |
| `meeting.scheduled` | New meeting scheduled |

## How It Will Work

1. Register a webhook URL in **Settings → Integrations**
2. Select which events to subscribe to
3. Flowxtra sends a `POST` request to your URL with event data
4. Verify the `X-Flowxtra-Signature` header for security
